# Scala Char abs()方法示例

> 原文:[https://www . geeksforgeeks . org/Scala-char-ABS-method-with-example/](https://www.geeksforgeeks.org/scala-char-abs-method-with-example/)

利用 **abs()** 方法求所述字符的绝对值。

> **方法定义:** def abs: Char
> 
> **返回类型:**返回 Char。

**例:1#**

```scala
// Scala program of abs()
// method

// Creating object
object GfG
{ 

    // Main method
    def main(args:Array[String])
    {

        // Applying abs() method 
        val result = 'a'.abs

        // Displays output
        println(result)

    }
} 
```

**Output:**

```scala
a

```
